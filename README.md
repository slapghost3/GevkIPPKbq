# 基于协同过滤算法的电影推荐系统 python92

## 项目概述

本项目是基于协同过滤算法的电影推荐系统，使用Python Flask作为后端服务框架，结合Vue.js前端框架，以及MySQL数据库进行数据的存储和管理。

## 技术栈

- 后端：Python Flask
- 前端：Vue.js
- 数据库：MySQL

## 运行环境

- Python 3.12
- MySQL 8.0

## 功能模块

### 用户端

- 用户注册登录、个人信息管理
 - 注册
 - 登录
 - 个人信息修改
- 电影浏览、分类查看、搜索功能、条件筛选
- 个性化推荐（协同过滤）、热门推荐
- 电影评分、收藏、评论
- 论坛交流讨论

### 管理端

- 电影信息管理
 - 添加
 - 编辑
 - 删除
- 用户账户管理
- 论坛、评论等管理
- 分类管理、系统配置
- 数据统计分析

## 系统角色

- 普通用户：可进行注册、登录、评论、评分等操作。
- 管理员：负责电影信息管理、用户账户管理、论坛和评论管理等。

## 目录结构

```
项目根目录
├── backend # 后端代码目录
├── frontend # 前端代码目录
└── database # 数据库相关文件
```

1. 安装Python 3.12环境
2. 安装MySQL 8.0数据库
3. 部署后端服务（Flask）
